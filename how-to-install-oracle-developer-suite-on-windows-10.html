How to install Oracle Developer Suite on Windows 10
Tue 08 May 2018 Oracle EBS Oracle EBS / Oracle Developer Suite / Windows 10

In spite of the fact that Oracle Developer Suite is certified with Windows 10, ODS installation that I did some time ago wasn't smooth. It required few tricks to make it work. Below you may find step by step description of the process. Note that I reference few Oracle Support resources in the post, to download them you'll need Oracle Support subscription.

Downloading

For the installation you will need

  • Base software media: Oracle Developer Suite 10g (10.1.2.0.2). You may download both disks from Oracle

  • Following patches from Oracle Support

    • Patch 10396165 WINDOWS 7 CERT:NEED AN INSTALL ONE OFF PATCH FOR IDS 10.1.2.0.2
    • Patch 5983622 Oracle Application Server 10g Release 2 (10.1.2) Patch Set 3 (10.1.2.

Installation

  • Oracle Installation utility is not able to check memory settings correctly and that's why it is required to explicitly specify minimum swap file size before the installation: Start Menu > click on Settings > Adjust the appearance and performance of Windows > Advanced > Change (Virtual memory section) > Uncheck the automatic setting, set the initial size to 256 MB and set maximum size to whatever you need (for example 8GB) and click Set. You may revert this change back after the installation.

  • Reboot your machine as Windows will request after the change

  • Run setup.exe delivered via Patch 10396165 with following command (via elevated command prompt)

    setup.exe -ignoreSysPrereqs

  • Click Next

  • Specify path to products.xml from Disk 1 from ODS archive (path should not contain any spaces) and click Next

  • Choose Complete Installation and click Next

  • Press Install and wait for installation completion

  • During the installation it will ask you where Disk 2 of ODS Base software media is located

Upgrading to ODS 10.1.2.3

  • If you performed all above steps correctly, now you should have ODS 10.1.2.0.2 installed, let's upgrade it to 10.1.2.3

  • Unpack content of Patch 5983622 and click on setup

  • We may run this installer without any command-line options, but it will prompt if we would like to ignore our OS version (choose y)

  • Choose same directory which was used during previous installation

  • Then choose 1st product

  • And click Install

Other References

There is no dedicated Oracle Support note describing installation process for Windows 10. Oracle suggests using one specified below. Please refer it for additional details.

  • Certification of Oracle Developer Suite 10g (10.1.2) on Windows 7 (32-bit) (Doc ID 1292919.1)
